How to Play

🎯 Goal

Rearrange the coloured blocks so that one of the stacks matches the Goal pattern shown on the right.

🖱️ Moving Blocks

  • Click — Click a top block to select it, then click any stack to move it there.
  • Drag & Drop — Drag a top block and drop it onto a target stack.

Only the top block of each stack can be moved.

🕹️ Game Modes

  • Basic — 5 blocks, random start and goal. Just solve the puzzle!
  • Advanced — Choose the number of blocks and design your own goal layout.
  • AI Mode — Turn-based battle vs the computer. First to reach the goal wins!

💡 Tips

  • Use empty stacks as temporary holding spots.
  • In AI Mode, watch the AI's moves to learn efficient strategies.
